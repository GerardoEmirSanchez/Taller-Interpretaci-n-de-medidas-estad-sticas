# Taller de Interpretación de medidas estadísticas y Análisis Estadístico para FRC

## Descripción
Este repositorio contiene el material completo del taller "Análisis Estadístico para FIRST Robotics Competition" diseñado para equipos de FRC que deseen implementar sistemas de scouting basados en datos 
objetivos y análisis estadístico.

## Contenido del Taller

### Documento Principal
- **`Taller__Interpretación_de_medidas_estadísticas.pdf`** - Material completo del taller
  
### Herramientas Prácticas
- **`Plantilla_Scouting.xlsx`** - Plantilla Excel con cálculos automáticos y semáforo de decisiones
